# 🤖 ARTEMIS v5.3 | Overseer Edition

Assistente multimodal de automação e suporte técnico via Telegram, desenvolvido para otimizar fluxos de trabalho em TI e monitoramento de infraestrutura.

## 🚀 Funcionalidades Principais

- **👁️ Visão Computacional:** Diagnóstico de prints de erro, hardware e leitura de documentos via Llama 3.2 Vision.
- **🎤 Interface Multimodal:** Transcrição de áudio (STT) e síntese de voz (TTS) para interação natural.
- **📊 Automação Office:** Geração dinâmica de documentos (.docx), planilhas (.xlsx) e apresentações (.pptx).
- **🛡️ Overseer System:** Monitoramento de hardware em tempo real (CPU/RAM) e gerenciamento de memória (Garbage Collection).
- **🎨 Geração de Imagens:** Criação de assets visuais técnicos via Pollinations AI.

## 🛠️ Stack Tecnológica

- **Linguagem:** Python 3.x
- **IA:** Groq API (Llama 3.3-70b & GPT-OSS 120B)
- **Bibliotecas:** `psutil`, `python-docx`, `openpyxl`, `gTTS`, `pyautogui`
- **Arquitetura:** Modular e Orientada a Eventos

## 🔧 Desafios de Engenharia & Resiliência
Durante o desenvolvimento, o projeto passou por manutenções críticas de **Depreciação de API**, migrando de modelos de preview para versões estáveis de produção. Isso garantiu o aprendizado prático em:
1. Gestão de dependências externas.
2. Tratamento de exceções para alta disponibilidade.
3. Manutenção corretiva e evolutiva.
