# 🤖 ARTEMIS v5.3 - Matrix Edition
Assistente Multimodal para Telegram integrada com IA de última geração.

## 🚀 Tecnologias
- **Groq API:** Llama 3.3 (Cérebro) e Whisper-large-v3 (Audição).
- **Python-Telegram-Bot:** Interface de comunicação.
- **Pollinations API:** Renderização de imagens.
- **gTTS:** Síntese de voz.

## 📊 Sistema Overseer
Inclui monitoramento em tempo real de CPU e RAM diretamente no terminal, com protocolos de auto-limpeza de cache.
